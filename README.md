# Polymarket vs. Binance options (Bitcoin thresholds)

Research code and analysis-ready data comparing **Polymarket** Yes prices to a **Black–Scholes** benchmark from **matched Binance** options.

- **Paper (LaTeX):** `paper/main.tex` (+ `paper/refs.bib`, figures under `paper/figures/` if present)
- **Data (CSV/JSON):** `data/` — see [`data/README.md`](data/README.md) for file descriptions and **source URLs**
- **Pipeline:** `run_research.py`, `paper/build_paper_artifacts.py`, `requirements.txt`

**Public repository:** https://github.com/victoriaportnaya/polymarketbinance

Upstream data remain subject to [Binance](https://data.binance.vision/) and [Polymarket](https://docs.polymarket.com/) terms.
